# Institutional Review Checklist

*Binding acceptance criteria — Design Manual Edition I · §41.1*

Every page must pass every criterion before release. A single failure blocks publication.

---

## I. The Standard (§4)

- [ ] Authentic — traceable to a sound source, represented as exactly what it is
- [ ] Beneficial — serves a real need, not a manufactured want
- [ ] Truthful — within the evidence, honest about limits
- [ ] Clinically responsible — no promise of cure, no risk of harm
- [ ] Worthy of the Sunnah — in substance and manner
- [ ] Inheritable — future generations would be proud, not ashamed
- [ ] Durable — would still be published in fifty years

## II. Architecture (§7–§8)

- [ ] One palette, one type system, one voice across all pages
- [ ] Masthead and footer identical everywhere (§31)
- [ ] Running head and folio on every content leaf (§8.5)
- [ ] URLs in institution terms — lowercase, hyphenated (§8.2)
- [ ] No department receives its own logo, colour, or navigation (§7.4)
- [ ] Content records carry all mandatory provenance fields (§8.3–§8.4)

## III. Colour (§13–§16)

- [ ] Only permitted palette tokens used — no introduced colours
- [ ] Only permitted ground pairings (§14.1)
- [ ] Gilt never on small body copy (§15.4)
- [ ] Ash never carries essential text (§15.4)
- [ ] Myrtle grounds rationed to gravity (§14.3)
- [ ] No gradients, drop shadows, traffic-light state colours (§16.3)

## IV. Typography (§17–§21)

- [ ] Cormorant · EB Garamond · Inter · Amiri only (§17.3)
- [ ] Type scale tokens respected — size, line-height, tracking (§18)
- [ ] Reading measure ≤ 34rem / ~66 characters (§19.1)
- [ ] Flush-left, ragged-right — no justified body (§19.2)
- [ ] Emphasis by italic, not bold, in reading text (§19.4)
- [ ] Arabic in Amiri, marked RTL, never letterspaced (§20)
- [ ] No exclamation marks (§10.4)

## V. Space & Layout (§22–§25)

- [ ] Spacing from the scale only — s1 through s8 (§22)
- [ ] Leaf padding clamp(s7 → s8) at desktop (§22.2)
- [ ] Corner radius 0 everywhere (§24.3)
- [ ] Depth by hairline and ground change only — no elevation (§24.2)
- [ ] Section order unchanged across breakpoints (§25.1 rule 2)
- [ ] Commerce beneath knowledge at every width (§25.1 rule 4)

## VI. Components (§26–§32)

- [ ] Seal used at permitted sizes; clear space honoured (§26)
- [ ] Quiet link is the default affordance (§28.1)
- [ ] Solid action in exactly one committed context per view (§28.3)
- [ ] Cards only where true peers; otherwise ruled rows (§29.1)
- [ ] Pathways styled as scholar cross-reference, never cross-sell (§29.5)
- [ ] Forms: real labels, no placeholder-as-label, canonical error strings (§30)
- [ ] Attestation apparatus always open, always present (§32.5)
- [ ] Source-marks keyboard and touch reachable (§32.3, §40.8)

## VII. Voice & Copy (§9–§12)

- [ ] No refused words (§10.2)
- [ ] No health-claim prohibition violations (§10.3)
- [ ] Institution action language — not marketing CTAs (§11)
- [ ] Grades capitalised: Established · Reported · Tried (§12.1)
- [ ] Canonical state strings where applicable (§38.5)

## VIII. Motion (§36–§37)

- [ ] Only catalogue motions used (§37.1)
- [ ] No bounce, parallax, scroll-jacking, auto-play (§37.3)
- [ ] `prefers-reduced-motion` honoured (§37.2)
- [ ] No content depends on motion to be readable (§40.6)

## IX. Accessibility (§40)

- [ ] WCAG 2.1 AA minimum; AAA where palette allows
- [ ] Colour never conveys information alone (§40.3)
- [ ] Visible Gilt focus ring on all interactives (§40.4)
- [ ] Targets ≥ 44×44px (§40.5)
- [ ] Full keyboard path including mobile nav (§40.6)
- [ ] One H1 per page; semantic landmarks (§40.7)

## X. Commerce & Ethics (§39)

- [ ] The Counter — not basket/cart (§39.1)
- [ ] Guest acquisition honoured (§39.4)
- [ ] No dark patterns — timers, scarcity, exit-intent, etc. (§39.5)
- [ ] Plain pricing, no surprise charges (§39.3)

## XI. Refused Trends (Appendix A)

- [ ] No drop shadows, glassmorphism, pill buttons, corner radii
- [ ] No gradient meshes, neon accents, kinetic type
- [ ] No shimmer skeletons, emoji, sticky pop-up CTAs
- [ ] No bento grids, corporate illustration, auto-playing heroes

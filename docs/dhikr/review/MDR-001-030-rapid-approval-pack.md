# MDR-001–030 — Rapid Scholarly Approval Pack

**Purpose**: a compact, single-pass working document for a scholar to record an initial decision per record — not a substitute for a full [MDR Scholarly Decision Record](MDR-SCHOLARLY-DECISION-RECORD-TEMPLATE.md), and not itself binding on the register. Each entry below summarises Stage 3B research; for full citations, evidence-tier detail, and the complete research trail, see the linked audit report. Protected Arabic (`fullArabicText`) is reproduced exactly as transcribed and is never edited from this pack.

**How to use this pack**: for each record, mark one decision, fill in the correction box only if proposing a change, and sign. A decision recorded here is a *draft recommendation* — it becomes binding only once transcribed into a completed decision record and committed via this project's standing checkpoint discipline (see docs/dhikr/40-scholarly-review-and-adjudication-framework.md, §J).

20 slots across 12 categories. The distribution (two slots for most categories, one each for Before Sleep / Upon Waking / Home / Mosque / Food and Drink) is a scheduling convenience for planning purposes — it is not a claim about how many authentic items exist for a given occasion. The scholarly reviewer may find more, fewer, or a different natural distribution once sourcing actually begins, and this plan should yield to that finding rather than the reverse.

## Sequencing recommendation

Source and review Morning and Evening remembrance first — the categories with the highest expected visitor relevance per the target user described in [01-product-scope.md](01-product-scope.md) — proceeding category by category rather than attempting all 20 slots in parallel. This allows the first published items to ship as soon as one complete, reviewed category exists, rather than waiting for the entire 20-slot set to clear review together.

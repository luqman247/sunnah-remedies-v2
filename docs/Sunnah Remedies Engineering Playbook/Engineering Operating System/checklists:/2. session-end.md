# Session End Checklist

> Purpose: Ensure every engineering session ends in a safe, verified, and documented state.

---

## Verification

- [ ] Build passes.

- [ ] Type-check passes.

- [ ] Lint passes.

- [ ] Development server tested.

- [ ] Browser verified.

- [ ] No console errors.

---

## Documentation

- [ ] Documentation updated.

- [ ] ADR updated (if required).

- [ ] README updated.

- [ ] Decision Log updated.

---

## Git

- [ ] Commit message prepared.

- [ ] Branch clean.

- [ ] Files reviewed.

- [ ] Ready to commit.

---

## Next Session

- [ ] Remaining tasks identified.

- [ ] Risks documented.

- [ ] Next milestone identified.

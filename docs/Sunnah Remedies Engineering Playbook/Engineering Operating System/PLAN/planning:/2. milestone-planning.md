# Milestone Planning Prompt

## Role

You are acting as the Programme Manager for the Sunnah Remedies platform.

---

## Objective

Break a large project into manageable engineering milestones.

---

## Inputs

- Engineering Handbook

- Enterprise Architecture

- Current Repository

- Business Goals

---

## Deliverables

Provide:

Executive Summary

Milestones

Objectives

Dependencies

Deliverables

Acceptance Criteria

Estimated Complexity

Risks

Success Criteria

---

## Rules

Each milestone should:

- Deliver business value.

- Be independently deployable.

- Have measurable outcomes.

- Minimise risk.

Do not implement code.

Produce a planning document only.

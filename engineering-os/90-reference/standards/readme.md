# AI Engineering Standards

> Purpose

The AI Engineering Standards define the principles, behaviours, and quality expectations for all AI-assisted engineering activities within the Sunnah Remedies platform.

These standards ensure that AI operates consistently, predictably, and in alignment with the Engineering Handbook.

---

# Philosophy

AI is an engineering collaborator.

AI should:

- Assist, not replace.
- Follow documented standards.
- Respect architecture.
- Stay within approved scope.
- Verify all work.
- Document decisions.
- Escalate uncertainty.

AI should strengthen engineering quality rather than simply increase development speed.

---

# Standards

This directory contains:

- Prompt Writing Standard
- Prompt Versioning Standard
- AI Collaboration Standard
- AI Quality Standard

Every AI interaction should follow these standards.

---

# Engineering Principles

AI should:

- Read documentation first.
- Understand before implementing.
- Reuse before creating.
- Verify before reporting.
- Explain reasoning.
- Update documentation.
- Respect architectural boundaries.
- Leave the repository in a better state.

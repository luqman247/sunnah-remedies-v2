# Knowledge Article Prompt

Create an internal engineering knowledge article.

Purpose:

Capture lessons learned, implementation patterns, troubleshooting guidance, or engineering best practices.

Include:

- Title
- Summary
- Problem
- Context
- Solution
- Code Examples (if appropriate)
- Common Mistakes
- Best Practices
- Related Modules
- Related ADRs
- Related Handbook Chapters
- Further Reading

Write for future engineers joining the project.

The article should preserve knowledge that is likely to be useful again.

## The three commitments

1. **Integrity** — accuracy and authenticity are never subordinate to commercial convenience.
2. **Longevity** — built to be maintained for decades.
3. **Excellence** — every surface reflects *iḥsān*: done beautifully because it is worth doing beautifully.

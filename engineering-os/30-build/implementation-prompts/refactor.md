# Refactor Prompt

Improve the internal quality of the implementation without changing observable behaviour.

Focus on:

- Readability
- Maintainability
- Reusability
- Performance
- Type Safety

Do NOT:

- Change UI unnecessarily
- Introduce new features
- Expand scope

Verify no regressions after completion.

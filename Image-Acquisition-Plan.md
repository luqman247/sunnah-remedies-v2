> Governed by `Photography-Art-Direction-Manual.md` and `Sunnah-Remedies-Design-Manual.md` §33–§35.

This plan lists **exactly** which photography each route requires, assigns a unique **Asset ID** to every frame, and supplies a **written brief** for the photographer. No asset enters production without an approved brief and ID.

**Current product state:** Many pages use gilt line studies as placeholders. This plan specifies the photography that **replaces** those placeholders when commissioned. Until commission, line studies remain; briefs are still binding for the shoot.

# §1 — Photography type definitions

Types used in this plan. **Lifestyle** and **flat lay** are redefined to match the Art Direction Manual — not industry defaults.

| Type | Definition | Reference | Refused alias |
|---|---|---|---|
| **Hero** | Primary page image; one subject; deep margin; sets subject-world | Aesop + British Museum | Full-bleed stock banner |
| **Editorial** | Supporting narrative image — room, desk, corridor, process | Monocle + Kinfolk | Magazine vitality |
| **Product** | Remedy as object — vessel, label legible, 4:5 vitrine | Aesop + Wellcome | E-commerce white void |
| **Lifestyle** | **Institutional context only** — person engaged in real work, not performing wellness | Harvard Exec Ed | Influencer wellness, stock smiles |
| **Architectural** | Building, facade, corridor, permanent structure | British Museum + Monocle | HDR postcard |
| **Clinical** | Consultation room, treatment bay, instruments at rest | Mayo Clinic | Fear/hero medical |
| **Pilgrimage** | Landscape, path, stone, lodge — 16:9 atmospheric | Kinfolk + British Museum | Travel brochure |
| **Laboratory** | Bench, glassware, balance, labelled samples | Wellcome Collection | Sci-fi lab |
| **Detail** | Macro — texture, script, seed surface, label | Wellcome specimen | Abstract bokeh |
| **Portrait** | Faculty/formal — plain ground, briefed, rare | Oxford college | Glamour headshot |
| **Specimen table** | Objects on table in vitrine logic — **not knolling flat lay** | Wellcome + Aesop | Instagram flat lay / knolling |
| **Texture** | Surface fill — linen, paper, stone — for text pages | Kinfolk | Fake marble backdrop |

§1.1 — **Flat lay is refused.** Where legacy briefs say "flat lay," this plan specifies **specimen table** (ordered, labelled, maximum three object types).

§1.2 — **Lifestyle is restricted.** Permitted only where this plan explicitly lists it. Otherwise use **hands**, **editorial**, or **no image**.

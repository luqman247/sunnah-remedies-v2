# Bug Fix Prompt

Your task is to resolve a defect while preserving existing behaviour.

Workflow:

Understand

↓

Reproduce

↓

Identify Root Cause

↓

Explain Root Cause

↓

Implement Minimal Fix

↓

Verify

↓

Regression Test

↓

Document

Never redesign unrelated systems while fixing bugs.

Deliver:

- Root Cause
- Files Changed
- Verification
- Regression Results

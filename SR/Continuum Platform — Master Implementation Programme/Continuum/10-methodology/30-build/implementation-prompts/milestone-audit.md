# Milestone Audit Prompt

Review the current milestone before implementation.

Read:

- Engineering Handbook
- Engineering Execution Plan
- Enterprise Architecture

Audit:

- Completed work
- Remaining work
- Technical debt
- Missing documentation
- Architecture compliance
- Risks

Produce:

- Progress Summary
- Missing Tasks
- Recommendations
- Updated Milestone Status

Do not implement code.

Audit only.

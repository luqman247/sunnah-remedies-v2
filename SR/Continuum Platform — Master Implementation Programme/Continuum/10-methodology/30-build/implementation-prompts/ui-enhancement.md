# UI Enhancement Prompt

Improve the existing interface without altering approved functionality.

Workflow:

Review Existing UI

↓

Understand Design System

↓

Implement Improvements

↓

Verify Browser

↓

Responsive Testing

↓

Accessibility Review

↓

Performance Review

Requirements:

- Preserve design language.
- Maintain consistency.
- Avoid unnecessary redesigns.
- Follow the Design System.
- Improve UX where appropriate.

Verification is mandatory before completion.

# AI Engineering Examples

> Purpose

The Examples library provides high-quality reference implementations for AI and human contributors.

Examples demonstrate what "good" looks like.

These examples should be treated as engineering reference material rather than templates to copy blindly.

---

## Philosophy

Rules explain standards.

Examples demonstrate standards.

Every example should:

- Follow the Engineering Handbook.
- Follow Repository Structure.
- Follow Naming Conventions.
- Follow Coding Standards.
- Follow Documentation Standards.
- Represent production-quality engineering.

## How to Use

Before asking AI to produce work:

1. Read the relevant standard.
2. Read a similar example.
3. Use both as context.

Examples improve consistency and quality.

## Engineering Principle

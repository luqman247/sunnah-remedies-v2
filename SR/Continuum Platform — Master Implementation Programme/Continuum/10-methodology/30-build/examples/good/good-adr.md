# Example ADR

ADR-004

Server First Architecture

---

Problem

Too much client-side JavaScript.

---

Decision

Adopt React Server Components wherever possible.

---

Benefits

Improved performance.

Improved SEO.

Smaller bundles.

---

Trade-offs

More server-side complexity.

---

Status

Accepted.

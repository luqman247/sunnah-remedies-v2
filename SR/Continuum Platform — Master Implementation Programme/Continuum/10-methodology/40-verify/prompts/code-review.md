# Code Review Prompt

Review the implementation without modifying it.

Evaluate:

- Readability
- Maintainability
- Reusability
- Strong typing
- Separation of concerns
- Module boundaries
- Security
- Performance
- Accessibility
- Documentation

Identify:

- Code smells
- Duplication
- Technical debt
- Potential bugs

Provide:

- Overall assessment
- Major findings
- Minor findings
- Recommendations

Do not implement changes.

Review only.

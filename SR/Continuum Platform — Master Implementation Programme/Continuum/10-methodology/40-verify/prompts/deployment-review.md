# Deployment Review Prompt

Review whether the implementation is ready for deployment.

Verify:

- Build passes.
- Documentation updated.
- Verification complete.
- Release Notes prepared.
- No blocking issues.

Check:

- Rollback strategy
- Environment variables
- Deployment risks
- Monitoring readiness

Result:

Ready

or

Not Ready

Explain why.

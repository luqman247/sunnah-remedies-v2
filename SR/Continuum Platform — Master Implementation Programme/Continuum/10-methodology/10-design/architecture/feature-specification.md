# Feature Specification Prompt

Design an implementation-ready specification for a new feature.

Include:

- Business Objective
- User Problem
- User Stories
- Functional Requirements
- Non-functional Requirements
- Technical Requirements
- Acceptance Criteria
- Risks
- Dependencies
- Future Enhancements

Every requirement should be measurable.

Do not implement the feature.

Produce specification only.

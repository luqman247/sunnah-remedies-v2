## 14.1 Master checklists (ready to sign)

Print, work through, sign, date. These are the authoritative operational checklists.

### Daily opening
- [ ] Alarm off; no overnight incident/leak/fault
- [ ] Fire exits clear; fire panel normal
- [ ] Lights/temperature/ventilation/scent to reset state
- [ ] Each public zone clean and at reset state
- [ ] Clinic rooms prepared to standard (if clinics today)
- [ ] Dispensary storage temperatures logged; handwashing stocked
- [ ] Reception tidy; systems and bookings up
- [ ] Today's appointments/courses/deliveries reviewed
- [ ] Team huddle held
- [ ] Opened · *Signed / time:* ______

### Daily closing
- [ ] No guest rushed; gentle close
- [ ] Clinic close-down: cleaned, waste/sharps secured, records filed
- [ ] Dispensary close-down: stock/temperatures logged, perishables secured, food areas cleaned
- [ ] Payments reconciled; no cash left out
- [ ] All zones reset for tomorrow
- [ ] Security sweep: doors/windows, hazards off, no one remaining
- [ ] Clean-desk: no PII/notes/keys left visible; confidential material locked
- [ ] Systems/screens locked
- [ ] Alarm on; locked · *Signed / time:* ______

### Onboarding (new colleague)
- [ ] Pre: workspace, least-privilege access, Handbook access, rota, buddy
- [ ] Day 1: welcome, HQ tour, **fire/first-aid/emergency briefing**, read Ch 00/01/02 + role chapter
- [ ] Week 1: role SOP walkthrough, shadowing, systems + data-protection basics, two-ledger + four-layers confirmed
- [ ] 30/60/90 check-ins; competence signed before unsupervised safety/clinical/scholarship work

### Offboarding
- [ ] Handover document written
- [ ] **All system access revoked same day**
- [ ] Keys/property/assets returned
- [ ] PII/records transferred to correct owner
- [ ] Exit conversation held

### Incident / accident report
- [ ] Made safe; people protected
- [ ] Responded (first aid / emergency services / containment)
- [ ] Recorded: what/when/who/action; external report if reportable
- [ ] Reported to lead (+ Standards Council if serious)
- [ ] Reviewed: root cause, controls updated, shared

### Goods-in (dispensary)
- [ ] Received against order; quantities/condition checked
- [ ] Provenance/certification present (reject if missing)
- [ ] Batch covered by / scheduled for lab verification
- [ ] Inspected for damage/tamper/spoilage/pests
- [ ] Recorded: date, quantity, batch/lot, expiry
- [ ] Stored in correct conditions

### Order fulfilment & packing
- [ ] Picked; item/grade/batch verified
- [ ] Quality-checked (condition, expiry, label)
- [ ] Packed to presentation standard with correct inserts
- [ ] Dispatch confirmed and triggered
- [ ] Recorded against order + batch

### Clinic room preparation (before every guest)
- [ ] Surfaces decontaminated to clinical standard
- [ ] Clean linens/coverings
- [ ] Single-use supplies stocked; sharps container present, not overfilled
- [ ] Correct waste containers available
- [ ] Hand hygiene/sanitiser ready
- [ ] Room private, calm, dignified

### Hijama session preconditions (all must be true)
- [ ] Practitioner trained + licensed/registered as required
- [ ] Local-authority special-treatment licence held
- [ ] Room to clinical-clean + IPC standard
- [ ] Informed consent obtained and recorded
- [ ] Screening + contraindications checked; no red flags
- [ ] Single-use sterile disposables + sharps container ready
- [ ] Emergency protocol + first-aider available

### Teaching studio preparation
- [ ] Set to session plan (layout/materials/AV)
- [ ] Current approved handouts/materials
- [ ] Registration ready
- [ ] Practical clinical teaching: IPC standard, single-use + sharps/clinical-waste, safety briefing
- [ ] Accessibility/comfort/prayer-break timings checked

### Journey duty-of-care pack (before every journey)
- [ ] Risk assessment refreshed
- [ ] Emergency protocol documented; staff briefed; next-of-kin reachable
- [ ] Insurance in force (confirmed with counsel)
- [ ] Safeguarding arrangements + named lead
- [ ] Staffing confirmed with responsibilities
- [ ] Every guest cleared, documented, consented
- [ ] Named contact reachable throughout

### Publish an article
- [ ] Every claim sourced; hadith graded/attributed; Qur'an verified
- [ ] Four layers distinguished
- [ ] Scholarly + Clinical (if health) + Editorial sign-off
- [ ] Safety line at point of decision (if health)
- [ ] Citations linked; images approved with alt text
- [ ] Provenance record complete; next-review date set
- [ ] Restraint check: adjectives cut

### Product launch
- [ ] Sourcing documented; lab verification recorded
- [ ] Clinical + Scholarly + Editorial sign-off
- [ ] Photography QC-passed, truthful, published; accuracy signed
- [ ] Stock live; no oversell; pricing honest and consistent
- [ ] Allergen/contraindication/medical-advice lines present
- [ ] Cross-links live; search indexed with honest availability
- [ ] Fulfilment + returns tested; regulatory flags cleared with counsel

### Photo QC (before publish)
- [ ] House grade matched; truthful (colour/scale/quantity)
- [ ] Sharp/exposed/white-balanced; retouch honest
- [ ] All crops/aspects present
- [ ] Metadata complete incl. alt text; rights/releases on file; licences unexpired
- [ ] Accuracy + (if sensitive) scholarly sign-off

## 14.4 Glossary

- **Adab** — right conduct; the character the work requires.
- **Amānah** — trust; the safe-keeping of what a guest entrusts to us.
- **Iḥsān** — excellence, doing the unseen as well as the seen.
- **Ṣidq** — truthfulness; describing without inflating.
- **Rifq / Ḥilm** — gentleness / forbearance.
- **Ṭahārah** — cleanliness/purity — a value and a clinical necessity.
- **The four layers** — established / interpreted / traditional / ours.
- **Two ledgers** — commercial and integrity; integrity holds veto.
- **The slow lane** — a permitted, unpenalised hold on any decision touching scholarship or safety until verified.
- **IPC** — infection prevention & control.
- **DAM** — digital asset management (the Media Library).
- **The Blueprint** — the prior-phase Operational Blueprint and its ten manuals.
- **Reset state** — how a zone must look when unattended.
- **Tier 1/2/3** — routine / judgement / precedent decisions.

## 14.5 Document control

- **Versioning:** this Handbook and every chapter are versioned (major = substantive change; minor = correction). Nothing overwritten; superseded versions archived with a superseded-by reference (mirrors Blueprint Doc 04).
- **Change process:** proposed changes go to the Handbook Custodian; substantive changes to standards, clinical, or scholarly content require the relevant Board and Standards Council approval before release.
- **Review:** full annual review; light quarterly review; immediate review after any incident or regulatory change.
- **Access:** internal classification; clinical and PII-related sections handled with appropriate confidentiality.
- **Ownership:** the Handbook Custodian (Office of the CXO) maintains the master; leads own their chapters.
- **Feedback:** anyone may propose an improvement — it is treated as a gift, not a complaint.

## 14.6 Closing word

This Handbook exists so that the standard survives any single person — so that a guest meets one honest, gentle, excellent institution whoever is on shift, and so that whoever inherits your role in ten years can carry it forward without ever having met you. That is the whole ambition: **build for the successor, and hold the Integrity Ledger above all.**

*End of the Institutional Operations Handbook.*
